## 🚀 Getting Started

### 1. Clone

```bash
git clone https://github.com/Arthur77dwq/quote-matey-v2.git
cd quote-matey-v2
```

### 2. Install

```bash
npm install
```

### 3. Run

```bash
npm run dev
```
